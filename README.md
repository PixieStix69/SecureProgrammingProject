# Secure Programming Project

Year 3, Semester 1 - Secure Programming Course

## Overview

This is our complete project for the Secure Programming Course. It contains three folders.

## Project Structure

- **Fixed Website/** - Vulnerable Flask web application for brute-force attack demonstration
- **Documentation/** - Project documentation and assignment materials
- **Images/** - Supporting images and screenshots

## Quick Start

1. Navigate to the `Fixed Website` directory
2. Install dependencies: `pip install flask flask-sqlalchemy requests`
3. Run the application: `python app.py`
